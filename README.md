## Installation

1. Open **Extensions** sidebar panel in VS Code (`Ctrl+Shift+X` or `Cmd+Shift+X`)
2. Search for `ANNO BLUE`
3. Click **Install**
4. Click **Reload** to reload your editor
5. Code > Preferences > Color Theme > **ANNO BLUE**
